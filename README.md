# Smart-Attendance-and-Task-Management-System
Python-based system for managing student attendance and academic tasks. It enables attendance tracking, student record management, task assignment, and progress monitoring, helping improve organization, accuracy, and efficiency in educational environments.


## Overview
A Python-based system that helps manage student attendance and tasks efficiently.

## Features
- Student attendance tracking
- Task management
- User-friendly interface
- Attendance reports

## Technologies Used
- Python
- SQLite


## Installation

1. Clone the repository
2. Install dependencies
3. Run the application
